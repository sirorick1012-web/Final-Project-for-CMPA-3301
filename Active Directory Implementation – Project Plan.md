## Phase 1: Initiation & Planning (Week 1)

1. Define project objectives, scope, and success criteria.
2. Conduct risk assessment and mitigation planning.

## Phase 2: Design (Week 2–3)  

1. Develop AD DS architecture (domain model, OU structure, naming conventions).
2. Group Policy structure.

## Phase 3: Implementation (Week 4–6)

 1. Create OU hierarchy
 2. Create security groups.
 3. Apply baseline Group Policies.
 
 ## Phase 4: Migration & Configuration (Week 7–8)
 
 1. Setup user and group accounts.
 2. Assign permissions.
 3. Test.
 4. Make adjustments as needed.
 
 ## Phase 5: Training & Handover (Week 9)
 
 1. Provide all documentation and SOPs.
 2. Qick recap and minor adjustments as needed.
 
 ## Phase 7: Post Creation Review (Week 10)
 
 1. Review performance and stability
 2. Document successes and failures for future use.
